# Git



### 1. Git이란?

* 분산 버전 관리 시스템
* 버전 : 컴퓨터 소프트웨어의 특정 상태
* 관리 : 어떤 일의 사무, 시설이나 물건의 유지,개량
* 프로그램 : 컴퓨터에서 실행될 때 특정 작업을 수행하는 일련의 명령어들의 모음

### Git 상태

* Untracked

  : 처음으로 관리되는 대상 --> 빨간색

  : Working Directory에 있는 상태 (Staging Area에 올라가기 전 상태)

* Tracked

  : 관리되고 있는 대상 (Staging Area에 올라간 상태)

  * modified --> 녹색
  * unmodified
